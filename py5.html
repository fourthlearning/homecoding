Array : อาร์เรย์ใช้เพื่อเก็บค่าหลายค่าในตัวแปรเดียว
เนื้อเรื่อง

ตัวอย่างการสร้าง Array : cars : ["Ford", "Volvo", "BMW"]
      อาร์เรย์เป็นตัวแปรพิเศษซึ่งสามารถเก็บค่าได้มากกว่าหนึ่งค่าในแต่ละครั้ง
หากคุณมีรายการสิ่งของ (เช่นรายชื่อรถยนต์) การจัดเก็บรถยนต์ในตัวแปรเดียวอาจมีลักษณะเช่นนี้:

car1 = "Ford"
car2 = "Volvo"
car3 = "BMW"
อย่างไรก็ตามถ้าคุณต้องการวนลูปผ่านรถยนต์และค้นหารถยนต์ที่เฉพาะเจาะจงล่ะ และถ้าคุณไม่มีรถยนต์ 3 คัน แต่เป็น 300
ทางออกคืออาร์เรย์!
อาร์เรย์สามารถเก็บค่าได้มากมายภายใต้ชื่อเดียวและคุณสามารถเข้าถึงค่าได้โดยอ้างอิงถึงหมายเลขดัชนี
หมายเหตุ: Python ไม่มีการสนับสนุนในตัวสำหรับ Arrays แต่สามารถใช้ Python Lists แทนได้

Array Method ที่ต้องจำ
Python has a set of built-in methods that you can use on lists/arrays.

# Method Description
1 append() เพิ่มองค์ประกอบที่ส่วนท้ายของรายการ
2 clear() เอาองค์ประกอบทั้งหมดออกจากรายการ
3 copy() คัดลอกรายการ
4 count() คืนค่าจำนวนองค์ประกอบที่มีค่าที่ระบุ
5 extend() เพิ่มองค์ประกอบของรายการ (หรือ iterable ใดๆ) ไปยังจุดสิ้นสุดของรายการปัจจุบัน
6 index() คืนค่าดัชนีขององค์ประกอบแรกที่มีค่าที่ระบุ
7 insert() เพิ่มองค์ประกอบที่ตำแหน่งที่ระบุ
8 pop() เอาองค์ประกอบออกตามตำแหน่งที่ระบุ
9 remove() ลบรายการแรกที่มีค่าที่ระบุ
10 reverse() กลับคำสั่งของรายการ
11 sort() เรียงลำดับรายการ
