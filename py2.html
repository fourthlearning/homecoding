Operator : ตัวดำเนินการ
ตัวดำเนินการทางด้านคณิตศาสตร์
# ตัวดำเนินการ ความหมาย ตัวอย่าง
1 + การบวก 5 + 2 = 7
2 - การลบ 5 - 2 = 3
3 * การคูณ 5 * 2
4 / การหาร 5 / 2 = 2.5
5 // หารไม่เอาเศษ เอาเฉพาะจำนวนเต็ม ทิ้งเศษ 5 // 2 = 2
6 % หารเอาเฉพาะเศษ 5 % 2 = 1
7 ** ยกกำลัง 5 ** 2 = 25
ตัวดำเนินการเปรียบเทียบ
# ตัวดำเนินการ ความหมาย ตัวอย่าง
1 == เท่ากัน 2 == 3 มีค่าเป็น false
2 != ไม่เท่ากัน 2 != 3 มีค่าเป็น True
3 > มากกว่า 2 > 3 มีค่าเป็น false
4 < น้อยกว่า 2 < 3 มีค่าเป็น True
5 >= มากกว่าหรือเท่ากับ 2 >= 3 มีค่าเป็น False
6 <= น้อยกว่าเท่ากับ 5 <= 2 มีค่าเป็น True
ตัวดำเนินการทางตรรกะ
# ตัวดำเนินการ ความหมาย ตัวอย่าง
1 and และ A and B
2 or หรือ A or B
3 not ไม่ A not B
ตัวดำเนินการกำหนดค่า
# ตัวดำเนินการ ตัวอย่าง ความหมาย
1 += x += 7 x = x + 7
2 -= x -= 7 x = x - 7
3 *= x *= 7 x = x * 7
4 /= x /= 7 x = x / 7
5 //= x //= 7 x = x // 7
6 %= x %= 7 x = x % 7
7 **= x **= 7 x = x ** 7
การรับค่า input
รูปแบบ
  • รับค่าทั่วไป : input()
  • รับค่าเป็นจำนวนเต็ม : int(input())
  • รับค่าเป็นทศนิยม : float(input())
  • และยังมีอีกหลายแบบ

ตัวอย่าง

ให้ A รับค่าเป็นจำนวนเต็มมันจะได้ A = int(input())

String Format
รูปแบบ หรือ แทนตัวแปร
  • %s – strings
  • %d – decimal integers (base-10)
  • %f – floating point display
  • %c – character
  • %b – binary
  • %o – octal
  • %x – hexadecimal with lowercase letters after 9
  • %X – hexadecimal with uppercase letters after 9
  • %e – exponent notation

ตัวอย่าง

print("My name is %s" %(name))
