Command Line (cmd)
วิธีการใช้เบื้องต้น

คำสั่ง Command Line พื้นฐาน และตัวอย่างการใช้คำสั่งต่างๆ

  • cd (Displays the name of or changes the current directory)
    • พิมพ์คำสั่ง cd เพื่อแสดง Current Directory
    พิมพ์คำสั่ง cd ตามด้วย Path จะเปลี่ยน Directory ปัจจุบันไปยัง Path นั้น เช่น
    C:\Users\User>cd Downloads
    C:\Users\User\Downloads>
  • mkdir (Creates a directory)
    • พิมพ์คำสั่ง mkdir ตามด้วยชื่อ Directory (หรือชื่อ Folder อย่างที่เราคุ้นเคยกัน) จะเป็นการสร้าง Directory ใหม่ขึ้นมา เช่น
    C:\Users\User>mkdir tempFolder
  • dir (Displays a list of files and subdirectories in a directory)
    • พิมพ์คำสั่ง dir เพื่อแสดงรายละเอียด File และ Directory ที่มีอยู่ใน Path ปัจจุบัน เช่น
    C:\Users\User>dir
    **สามารถเพิ่ม tag /b เพื่อให้แสดงเฉพาะชื่อ File หรือชื่อ Directory ได้ เช่น
    C:\Users\User>dir /b
  • ls แสดงชื่อไฟล์
    • C:\Users\User\Downloads> ls
